# 🚀 Inicio Rápido

## 3 Pasos para Empezar

### 1️⃣ Instalar Dependencias
```bash
cd construccion-admin
npm install
```

### 2️⃣ Iniciar la Aplicación
```bash
npm run dev
```

### 3️⃣ Abrir en el Navegador
```
http://localhost:3000
```
